## Git / GitHub 사용 내역

본 프로젝트는 Git과 GitHub를 활용한 협업 방식으로 진행되었습니다.
각 팀원은 개별 브랜치를 생성하여 기능을 구현한 후,
Pull Request를 통해 변경 사항을 공유하고 병합하였습니다.

브랜치 전략: main / ayra-feature / feature/samsung-page-footer

개발 도구: Visual Studio Code
